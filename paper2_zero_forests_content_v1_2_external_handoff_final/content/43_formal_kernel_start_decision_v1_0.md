# Formal kernel start decision v1.0

The first formal-kernel target is the Schwarzschild common-denominator algebra leaf:

```text
2*M/(r**2*(r-2*M)) - 2*M/(r**2*(r-2*M)) = 0
under: r**2*(r-2*M) != 0
```

FLRW exponential cancellation is second, after branch/domain assumptions are made more formal.
